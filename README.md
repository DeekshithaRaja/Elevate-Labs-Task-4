SHORT DESCRIPTION ABOUT THE PROJECT:


This project focuses on converting an existing desktop-only webpage into a mobile-friendly design using CSS media queries. We used the landing page created in Task 1 as the base and modified its layout to make it responsive across different screen sizes. The design was updated by stacking elements vertically, adjusting font sizes, and improving spacing for smaller devices. The navigation bar was also adjusted to display properly on mobile screens, and images were made responsive to fit within their containers. Media queries were implemented to apply styles based on screen width, ensuring a smooth user experience. The responsiveness was tested using browser tools to verify proper behavior on mobile viewports.
